# 서울 미세먼지 예보 앱 🌫️

서울 지역의 미세먼지(PM10) 및 초미세먼지(PM2.5) 예보를 확인할 수 있는 모바일 웹 애플리케이션입니다.

## 주요 기능

- 📱 모바일 최적화 디자인
- 🌫️ 미세먼지(PM10) 오늘/내일 예보
- 🌪️ 초미세먼지(PM2.5) 오늘/내일 예보
- 🔄 자동 새로고침 (4시간마다)
- 💾 로컬 데이터 캐싱
- 🎨 등급별 색상 구분 (좋음/보통/나쁨/매우나쁨)

## 사용 방법

### 1. API 키 발급

1. [공공데이터포털](https://www.data.go.kr) 회원가입
2. "에어코리아 대기오염정보" 검색
3. 활용신청 후 API 키 발급 (승인까지 1-2시간 소요)

### 2. 앱 실행

1. 웹 브라우저에서 앱 접속
2. 설정 버튼(⚙️) 클릭
3. 발급받은 API 키 입력
4. 저장 후 자동으로 데이터 로드

### 3. 홈 화면에 추가 (PWA)

**iOS (Safari):**
1. 공유 버튼 탭
2. "홈 화면에 추가" 선택

**Android (Chrome):**
1. 메뉴(⋮) 탭
2. "홈 화면에 추가" 선택

## 기술 스택

- AirKorea API
